# 프로젝트 컨벤션

## 브랜치 네이밍
- `feat/#N-brief-description` — 새 기능
- `fix/#N-brief-description` — 버그 수정
- `chore/#N-brief-description` — 설정, 의존성 등

## 커밋 메시지
```
feat/#3-add-repo-command: 레포 등록 명령어 추가
fix/#5-timezone-bug: KST 시간대 변환 오류 수정
chore/#2-update-deps: 의존성 버전 업데이트
```

## 슬래시 커맨드
- `/feat <요구사항>` — 기능 브랜치 생성 + 개발 계획 작성
- `/commit [메모]` — 변경사항 분석 후 컨벤션에 맞는 커밋 메시지 생성 및 커밋
